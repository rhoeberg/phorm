# phorm
Prototype of a nodebased tool for generative and reactive visuals with a focus on live performances and VJ setups.

(under development and not yet suitable for actual use)

---
### Dependencies
- PortAudio
- glew
- glm
- tiny file dialogs
- imgui
- stb libs
